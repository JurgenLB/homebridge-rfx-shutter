> Boilerplate lib.

### Setup

1. Fork the project or manually run:
```bash
git clone git@github.com:boilerz/boilerplate-lib.git my-lib
cd my-lib
git remote set-url origin git@github.com:gh_user/my-lib
```

Then after pull run:

```bash
yarn install
yarn custom
```
